1. 偏导数基本概念▼

偏导数本质上就是一元导数——对哪个变量求导时，其他变量当常数。

定义
\[\frac{\partial z}{\partial x}\bigg|_{(x_0,y_0)} = \lim_{\Delta x\to 0}\frac{f(x_0+\Delta x, y_0)-f(x_0,y_0)}{\Delta x}\]\[\frac{\partial z}{\partial y}\bigg|_{(x_0,y_0)} = \lim_{\Delta y\to 0}\frac{f(x_0, y_0+\Delta y)-f(x_0,y_0)}{\Delta y}\]
常见错误：∂z/∂x ≠ dz/dx。偏导数记号∂不是除法，不能随便约分！
设 z=x²y+sin(xy)，求 ∂z/∂x 和 ∂z/∂y。
  1. 对 x 求偏导：y 视为常数 → ∂z/∂x = 2xy + y·cos(xy)。
  2. 对 y 求偏导：x 视为常数 → ∂z/∂y = x² + x·cos(xy)。
2. 复合函数链式法则 必考▼

链式法则是多元微分最核心的技巧。画变量依赖树永远不会错。

标准链式法则（z=f(u,v), u=u(x,y), v=v(x,y)）
\[\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u}\cdot\frac{\partial u}{\partial x} + \frac{\partial f}{\partial v}\cdot\frac{\partial v}{\partial x}\]\[\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u}\cdot\frac{\partial u}{\partial y} + \frac{\partial f}{\partial v}\cdot\frac{\partial v}{\partial y}\]
口诀：每条路径上偏导相乘，所有路径结果相加。z到x有两条路（z→u→x 和 z→v→x），所以是两个乘积之和。
设 \(z = f\left(xy,\; \frac{x}{y}\right) + \sin y\)，求 \(\frac{\partial z}{\partial x}\) 和 \(\frac{\partial^2 z}{\partial x\partial y}\)。
（23-24真题计算题1）
  1. 令 u=xy，v=x/y，则 z=f(u,v)。
  2. uₓ=y, u_y=x；vₓ=1/y, v_y=-x/y²。
  3. ∂z/∂x = f₁'·y + f₂'·(1/y)。
  4. 对y再求导：∂/∂y(f₁'·y) = (f₁₁''·x + f₁₂''·(-x/y²))·y + f₁'·1。
  5. ∂/∂y(f₂'·(1/y)) = (f₂₁''·x + f₂₂''·(-x/y²))·(1/y) + f₂'·(-1/y²)。
  6. 利用f₁₂''=f₂₁''合并同类项。
∂z/∂x = y·f₁' + (1/y)·f₂'；∂²z/∂x∂y 含 f₁₁'', f₁₂'', f₂₂'' 表达式
设 z = sin(u²+v)，u=xy，v=x+y，求 ∂z/∂x。
  1. ∂z/∂u = 2u·cos(u²+v)，∂z/∂v = cos(u²+v)。
  2. uₓ = y，vₓ = 1。
  3. ∂z/∂x = 2u·cos(u²+v)·y + cos(u²+v)·1 = cos(u²+v)·(2uy+1)。
  4. 代回 u=xy, v=x+y：∂z/∂x = cos(x²y²+x+y)·(2xy²+1)。
3. 隐函数求偏导 必考▼
隐函数存在定理（F(x,y,z)=0 确定 z=z(x,y)）
\[F(x,y,z)=0,\quad F_z\neq 0\;\Longrightarrow\;\frac{\partial z}{\partial x}=-\frac{F_x}{F_z},\;\frac{\partial z}{\partial y}=-\frac{F_y}{F_z}\]
最容易丢分的地方：忘记负号！记住是 -Fₓ/F_z，不是 Fₓ/F_z。

\(\dfrac{yz}{z^2-xy}\)
