📘 Day 1 · 概率基础与一维分布

核心：贝叶斯公式 + 一维常用分布 + 分布函数

一、随机事件与概率

核心公式

P(B|A) = P(AB)/P(A)   (P(A) > 0)
P(AB) = P(A)P(B|A) = P(B)P(A|B)   乘法公式
P(B) = ∑i P(Ai)P(B|Ai)   全概率公式
P(Aj|B) = P(Aj)P(B|Aj)∑i P(Ai)P(B|Ai)   贝叶斯公式

⚡ 贝叶斯公式 — 必考大题计算模板

三步法：

  1. 设事件：A1,A2,...,An 为原因，B 为结果
  2. 写先验：P(Ai) = 各原因占比
  3. 写似然：P(B|Ai) = 各原因下结果发生概率
  4. 代贝叶斯：P(Aj|B) = 分子(P(Aj)P(B|Aj)) / 分母(全概率)
📝 例题1（贝叶斯 · 三厂次品）

某厂三条生产线：甲占50%次品率2%，乙占30%次品率3%，丙占20%次品率5%。随机抽一件是次品，求来自甲的概率。

解：设 A1,A2,A3 = 甲/乙/丙，B = 次品

P(A1)=0.5, P(A2)=0.3, P(A3)=0.2

P(B|A1)=0.02, P(B|A2)=0.03, P(B|A3)=0.05

全概率：P(B) = 0.5×0.02+0.3×0.03+0.2×0.05 = 0.029

贝叶斯：P(A1|B) = 0.5×0.02 / 0.029 ≈ 0.3448（约34.48%）

二、一维随机变量

分布函数 F(x) = P(X ≤ x)

性质：

  • 0 ≤ F(x) ≤ 1，F(−∞)=0，F(+∞)=1
  • F(x) 单调不减
  • F(x) 右连续
  • P(a < X ≤ b) = F(b) − F(a)

📊 常用分布一览

分布记号密度/分布律E(X)D(X)
0-1B(1,p)P(X=1)=ppp(1−p)
二项B(n,p)C(n,k)pᵏ(1−p)n−knpnp(1−p)
泊松P(λ)λᵏe−λ/k!λλ
均匀U[a,b]1/(b−a), a≤x≤b(a+b)/2(b−a)²/12
指数Exp(λ)λe−λx, x≥01/λ1/λ²
正态N(μ,σ²)(1/√(2π)σ)·exp(−(x−μ)²/2σ²)μσ²
📝 例题2（连续密度求概率）

X 的密度 f(x)=2x (0≤x≤1)，求 P(0.3<X<0.7)。

P(0.3<X<0.7) = ∫0.30.7 2x dx = [x²]0.30.7 = 0.49 − 0.09 = 0.4

📝 例题3（分布函数）

X 密度 f(x)=1/2 (0≤x≤2)，求分布函数 F(x)。

x<0：F(x)=0

0≤x≤2：F(x)=∫0x (1/2)dt = x/2

x>2：F(x)=1

即 F(x) = { 0, x<0; x/2, 0≤x≤2; 1, x>2 }

📗 Day 2 · 多维分布与数字特征

核心：二维联合分布 + 边缘/条件分布 + 协方差 + 中心极限定理

一、二维随机变量

核心公式

边缘密度：fX(x) = ∫−∞+∞ f(x,y) dy
条件密度：fY|X(y|x) = f(x,y) / fX(x)
独立 ⇔ f(x,y) = fX(x)·fY(y)（连续）
独立 ⇔ pij = pi··p·j（离散）
📝 例题4（边缘密度）

(X,Y) 联合密度 f(x,y)=2 (0≤x≤y≤1)，求 fX(x)。

fX(x) = ∫x1 2 dy = 2(1−x)，0≤x≤1；其他为0

📝 例题5（Z=max(X,Y)）

(X,Y)联合分布律：P(0,0)=0.1, P(0,1)=0.2, P(0,2)=0.1, P(1,0)=0.2, P(1,1)=0.1, P(1,2)=0.2, P(2,2)=0.1。求 Z=max(X,Y) 分布律和期望。

Z=0：(0,0) → P=0.1

Z=1：(0,1),(1,0),(1,1) → P=0.2+0.2+0.1=0.5

Z=2：(0,2),(1,2),(2,2) → P=0.1+0.2+0.1=0.4

校验：0.1+0.5+0.4=1 ✓

E(Z) = 0×0.1 + 1×0.5 + 2×0.4 = 1.3

二、数字特征

期望、方差、协方差

E(X) = ∑ xipi（离散）  |  E(X) = ∫ x·f(x)dx（连续）
D(X) = E(X²) − [E(X)]²   ← 计算方差首选公式
Cov(X,Y) = E(XY) − E(X)E(Y)
ρXY = Cov(X,Y) / √[D(X)D(Y)]   (−1 ≤ ρ ≤ 1)

期望与方差性质

E(aX+b) = aE(X)+b
E(X+Y) = E(X)+E(Y)   ← 无条件成立！
D(aX+b) = a²D(X)   ← 常数b不影响方差
D(X+Y) = D(X)+D(Y)+2Cov(X,Y)

⚠️ 真题陷阱：D(2X+3) = 4D(X)，不是 2D(X)+3！

三、中心极限定理

⚡ 中心极限定理 — 应用题必考

(∑Xi − nμ) / (√n·σ) ≈ N(0,1)   (n 充分大)

解题步骤（供电问题模板）：

  1. 设 Xi = 第i台用电量，求 μ = E(Xi), σ² = D(Xi)
  2. 总用电 Sn = ∑Xi ≈ N(nμ, nσ²)
  3. 标准化：P(Sn ≤ C) ≈ Φ((C−nμ)/(√n·σ))
  4. 查表得概率
📝 例题6（中心极限定理）

100台车床，每台开工率0.7且独立。每台耗电1kW，供电80kW，求供电不足概率。（已知 Φ(2.18)=0.9854）

Xi ∼ B(1,0.7)，E=0.7，D=0.21

S ∼ N(70, 21)，P(S>80) = P(Z > 10/√21) = P(Z > 2.18) = 1−0.9854 = 0.0146（1.46%）

📙 Day 3 · 统计推断与参数估计

核心：抽样分布 + 矩估计 + 极大似然估计 + 无偏性

一、抽样分布

样本均值与样本方差

样本均值：X̄ = (1/n)·∑i Xi
样本方差：S² = [1/(n−1)]·∑i (Xi − X̄)²   ← 分母 n−1！

三大抽样分布

分布定义用途
χ²分布(n−1)S²/σ² ∼ χ²(n−1)方差检验
t分布(X̄−μ)/(S/√n) ∼ t(n−1)σ²未知时均值估计
F分布(S₁²/σ₁²)/(S₂²/σ₂²) ∼ F(n₁−1, n₂−1)两方差比较

记忆口诀：χ²管方差，t管均值，F管两方差比

二、参数估计

⚡ 极大似然估计四步法

第一步：写似然函数 L(θ) = ∏ f(xi;θ)

第二步：取对数 ln L(θ) = ∑ ln f(xi;θ)

第三步：求偏导 ∂(ln L)/∂θ = 0

第四步：解出 θ̂

📝 例题7（指数分布MLE）

X1,...,Xn 来自 f(x;λ)=λe−λx (x>0)，求 λ 的MLE。

L(λ) = λn·e−λ∑xi

ln L = n·ln λ − λ·∑xi

d(ln L)/dλ = n/λ − ∑xi = 0

λ̂ = n/∑xi = 1/X̄

📝 例题8（正态分布MLE）

X1,...,Xn ∼ N(μ,σ²)，求 μ,σ² 的MLE。

∂lnL/∂μ = 0 → μ̂ = X̄

∂lnL/∂σ² = 0 → σ̂² = (1/n)·∑(Xi−X̄)²

⚠️ σ̂² 是有偏的！无偏估计分母用 n−1

无偏估计与有效性

E(θ̂) = θ  ⇔  θ̂ 是无偏估计
若 E(θ̂₁) = E(θ̂₂) = θ，且 D(θ̂₁) < D(θ̂₂)  ⇒  θ̂₁ 更有效

💡 解题锦囊

考场技巧 · 失分点 · 条件→方法速查

🏆 考场技巧 Top 5

  1. 先易后难：选择题前5题简单先拿分；贝叶斯和参数估计是送分题优先做
  2. 分布函数分段写：x<a、a≤x≤b、x>b 三段，少一段扣分
  3. 中心极限：必须写"由中心极限定理"，不写可能扣分
  4. 极大似然：一定要取对数！直接求 L(θ) 最大值很难
  5. 检查归一性：求完常数验证 ∫f(x)dx=1 或 ∑pi=1

⚠️ 常见失分点

失分点❌ 错误✅ 正确
分布函数分段只写0≤x≤1段写全 x<0, 0≤x≤1, x>1
方差性质D(2X+3)=2D(X)+3D(2X+3)=4D(X)
样本方差分母S²用分母n无偏估计分母n−1
极大似然直接对L(θ)求导先取对数再求导
贝叶斯分母分母只加一项分母加所有P(Ai)P(B|Ai)
独立≠不相关只看E(XY)=E(X)E(Y)需验证联合=边缘乘积
正态标准化Z=(X−μ)/σ²Z=(X−μ)/σ（除以σ!）
