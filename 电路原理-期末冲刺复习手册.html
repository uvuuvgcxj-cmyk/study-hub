⚡ Day 1 — 直流电路 + 动态电路

电路分析的两大基石：基尔霍夫定律 + 等效变换

一、节点电压法

核心思想

选择一个参考节点（通常接地），对其余 $n-1$ 个节点列 KCL 方程。每个节点电压即为该节点相对参考点的电位。适用于节点少、支路多的电路。

标准步骤（四步法）：

① 选参考节点（通常选下方公共节点），标注其余节点电压 $V_1, V_2, \ldots$

② 对每个非参考节点列 KCL：$\sum \frac{V_i - V_j}{R_{ij}} = \sum I_{source}$（流出 = 注入）

③ 整理为矩阵形式：$G_{11}V_1 + G_{12}V_2 + \cdots = I_{S11}$

④ 解线性方程组求各节点电压，再由节点电压求支路电流

关键概念

自电导 $G_{ii}$：与节点 $i$ 直接相连的所有支路电导之和，恒为正。

互电导 $G_{ij}$：节点 $i$ 与 $j$ 之间的公共电导，取负值。若两节点无直接连接，互电导为零。

等效电流源 $I_{Sii}$：流入节点 $i$ 的独立电流源代数和（流入为正，流出为负）。电压源需先做电源等效变换或用超节点法。

📝 真题详解：节点电压法求各支路电流

如上图，$R_1=2\Omega$, $R_2=4\Omega$, $R_3=2\Omega$, $V_{S1}=12V$, $V_{S2}=6V$，求节点电压 $V_1$ 及各支路电流。

✅ 完整解答

第一步：列节点1的KCL方程（流出=0）

$$\frac{V_1 - V_{S1}}{R_1} + \frac{V_1 - 0}{R_2} + \frac{V_1 - V_{S2}}{R_3} = 0$$

第二步：代入数值

$$\frac{V_1 - 12}{2} + \frac{V_1}{4} + \frac{V_1 - 6}{2} = 0$$

第三步：通分（乘以4）

$$2(V_1-12) + V_1 + 2(V_1-6) = 0$$

$$2V_1 - 24 + V_1 + 2V_1 - 12 = 0$$

第四步：求解

$$5V_1 = 36 \Rightarrow \boxed{V_1 = 7.2V}$$

第五步：求各支路电流

$I_1 = \frac{V_1 - V_{S1}}{R_1} = \frac{7.2 - 12}{2} = -2.4A$（负号表示实际方向向上）

$I_2 = \frac{V_1}{R_2} = \frac{7.2}{4} = 1.8A$（向下）

$I_3 = \frac{V_1 - V_{S2}}{R_3} = \frac{7.2 - 6}{2} = 0.6A$（向下）

验证KCL： $-I_1 = I_2 + I_3 \Rightarrow 2.4 = 1.8 + 0.6 = 2.4$ ✓

二、网孔电流法

核心思想

对每个独立网孔设定顺时针网孔电流，沿网孔列 KVL 方程。适用于网孔少、节点多的电路。与节点电压法互补——哪个变量少就用哪个。

标准步骤（四步法）：

① 设定各网孔电流方向（通常都选顺时针）

② 对每个网孔列 KVL：$\sum R_{ii} I_i - \sum R_{ij} I_j = \sum V_{source}$

③ 自电阻 $R_{ii}$ = 该网孔所有电阻之和（正）；互电阻 $R_{ij}$ = 相邻网孔共享电阻（负）

④ 电压源沿网孔电流方向为正（电压升为正），解方程组

📝 真题详解：网孔电流法求各支路电流

如上图，$R_1=4\Omega$, $R_2=2\Omega$, $R_3=4\Omega$, $R_4=2\Omega$, $R_5=4\Omega$, $V_{S1}=20V$, $V_{S2}=10V$，求网孔电流 $I_1, I_2$。

✅ 完整解答

网孔1（左，I₁ 顺时针）：

自电阻：$R_{11} = R_1 + R_3 + R_4 = 4+4+2 = 10\Omega$

互电阻：$R_{12} = -R_3 = -4\Omega$（R₃ 为共享）

$$10I_1 - 4I_2 = V_{S1} = 20 \quad \cdots(1)$$

网孔2（右，I₂ 顺时针）：

自电阻：$R_{22} = R_2 + R_3 + R_5 = 2+4+4 = 10\Omega$

互电阻：$R_{21} = -R_3 = -4\Omega$

$$-4I_1 + 10I_2 = -V_{S2} = -10 \quad \cdots(2)$$

注意 Vs₂ 方向：I₂ 顺时针经过 Vs₂ 时从+到−，是电压降，所以取负。

联立求解：由(1)得 $I_1 = 2 + 0.4I_2$，代入(2)：

$$-4(2+0.4I_2) + 10I_2 = -10$$

$$-8 - 1.6I_2 + 10I_2 = -10 \Rightarrow 8.4I_2 = -2$$

$$\boxed{I_2 \approx -0.238A, \quad I_1 = 2 + 0.4(-0.238) \approx 1.905A}$$

R₃ 支路电流： $I_{R3} = I_1 - I_2 = 1.905 - (-0.238) = 2.143A$（向下）

⚠️ 含电流源支路的处理（超网孔法）：

如果电流源 $I_S$ 位于两个网孔的公共支路上，不要对这两个网孔分别列KVL，而是：

① 把两个网孔合并为一个"超网孔"（去掉电流源支路）

② 对超网孔列一个KVL方程

③ 补充约束方程：$I_2 - I_1 = I_S$（或 $I_1 - I_2 = I_S$，取决于方向）

④ 两个方程两个未知数，求解

三、叠加定理

核心思想

线性电路中，多个独立源共同作用时的响应 = 每个独立源单独作用时响应的代数和。这是线性电路最基本的性质。

图1.3 叠加定理分解 — 左：原电路 / 右上：Vs₁单独(Vs₂短路) / 左下：Vs₂单独(Vs₁短路)

⚠️ 叠加定理四大规则：

① 电压源不作用 → 短路（$V_S = 0$，用导线代替）

② 电流源不作用 → 开路（$I_S = 0$，断开）

③ 受控源不能单独作用，但每次都要保留在电路中（它不是激励）

④ 叠加只适用于电压和电流，不适用于功率（$P=I^2R$ 是平方关系，$(I_1+I_2)^2 \neq I_1^2 + I_2^2$）

四、戴维南定理与诺顿定理

戴维南等效

任何线性含源二端网络都可以等效为一个电压源 $V_{th}$ 串联电阻 $R_{th}$。

戴维南三步法：

第一步：求开路电压 $V_{oc} = V_{th}$ —— 负载开路，用节点法/网孔法/叠加法求端口电压

第二步：求等效电阻 $R_{th}$

　• 无受控源：独立源置零后，从端口看进去的串并联等效电阻

　• 有受控源：外加电源法（端口加 $V_0$，独立源置零但保留受控源，求 $I_0$，$R_{th}=V_0/I_0$）

　　或开路短路法：$R_{th} = V_{oc}/I_{sc}$

第三步：画等效电路：$V_{th}$ 串联 $R_{th}$，接上负载 $R_L$

图1.4 戴维南等效 — 任何线性含源网络都可化简为 V_th 串 R_th

诺顿等效

任何线性含源二端网络也可以等效为一个电流源 $I_N$ 并联电阻 $R_N$。诺顿定理与戴维南定理是对偶关系。

转换关系：$I_N = I_{sc} = V_{th}/R_{th}$，$R_N = R_{th}$（等效电阻相同）。

图1.5 诺顿等效 — 电流源 I_N 并联 R_N，与戴维南互为对偶
📝 真题详解：戴维南等效求负载最大功率

电路中 $V_S=10V$, $R_1=4\Omega$, $R_2=6\Omega$，负载 $R_L$ 接在 $R_1$ 两端。求 $R_L$ 获得最大功率时的值及最大功率。

✅ 完整解答

① 求开路电压 $V_{oc}$（去掉 $R_L$）：

开路时 $R_1$ 和 $R_2$ 串联分压：

$$V_{oc} = V_S \times \frac{R_1}{R_1+R_2} = 10 \times \frac{4}{4+6} = 4V = V_{th}$$

② 求等效电阻 $R_{th}$（电压源短路）：

从端口看进去，$R_1$ 和 $R_2$ 并联：

$$R_{th} = R_1 \parallel R_2 = \frac{4 \times 6}{4+6} = 2.4\Omega$$

③ 最大功率传输条件： $R_L = R_{th} = 2.4\Omega$

$$P_{max} = \frac{V_{th}^2}{4R_{th}} = \frac{4^2}{4 \times 2.4} = \frac{16}{9.6} \approx 1.67W$$

五、动态电路（一阶 RC / RL）

换路定则

换路瞬间（$t=0$），电容电压和电感电流不能突变（能量连续性）：

$$u_C(0_+) = u_C(0_-), \quad i_L(0_+) = i_L(0_-)$$

注意：虽然 $u_C$ 和 $i_L$ 不能突变，但电容电流 $i_C$ 和电感电压 $u_L$ 可以突变！

因为 $i_C = C\frac{du_C}{dt}$，$u_L = L\frac{di_L}{dt}$，它们取决于变化率而非瞬时值。

三要素法（必考重点）

一阶电路任意变量 $f(t)$ 的全响应：

$$\boxed{f(t) = f(\infty) + [f(0_+) - f(\infty)]e^{-t/\tau}}$$

三要素详解：

① $f(0_+)$ — 初始值：由换路定则确定 $u_C(0_+)$ 或 $i_L(0_+)$，再画 0+ 等效电路（电容用电压源 $u_C(0_+)$ 替代，电感用电流源 $i_L(0_+)$ 替代）求其他变量

② $f(\infty)$ — 稳态值：换路后电路达到新稳态（$t\to\infty$），此时电容开路、电感短路

③ $\tau$ — 时间常数：

　• RC电路：$\tau = R_{eq}C$

　• RL电路：$\tau = L/R_{eq}$

　• $R_{eq}$ 是从储能元件两端看进去的等效电阻（独立源置零后）

🌊 Day 2 — 正弦稳态 + 耦合电感 + 三相电路

相量法是核心工具，三相电路必考大题

一、相量法基础

正弦量与相量的转换

正弦电压 $u(t) = U_m\cos(\omega t + \varphi)$ 对应的有效值相量为：

$$\dot{U} = U\angle\varphi = U e^{j\varphi}$$

其中 $U = U_m / \sqrt{2}$ 为有效值，$\varphi$ 为初相，$\omega = 2\pi f$ 为角频率。

相量法的本质：把时域的微分/积分运算转化为复数域的乘除运算。

• 微分：$\frac{di}{dt} \to j\omega \dot{I}$（乘以 $j\omega$）

• 积分：$\int i\,dt \to \frac{\dot{I}}{j\omega}$（除以 $j\omega$）

• KCL/KVL 在相量域形式不变：$\sum \dot{I} = 0$，$\sum \dot{U} = 0$

阻抗与导纳

元件阻抗 $Z$导纳 $Y=1/Z$电压电流相位关系
电阻 R$R$$1/R = G$同相（φ=0）
电感 L$j\omega L$$1/(j\omega L)$电压超前电流 90°
电容 C$1/(j\omega C)$$j\omega C$电压滞后电流 90°
串联总阻抗$Z = Z_1 + Z_2 + \cdots = R + jX$
并联总导纳$Y = Y_1 + Y_2 + \cdots = G + jB$

阻抗的一般形式：$Z = R + jX$，其中 $R$ 为电阻，$X$ 为电抗。$|Z| = \sqrt{R^2+X^2}$，$\varphi = \arctan(X/R)$。

判别：$X > 0$ 感性（电压超前电流）；$X < 0$ 容性（电压滞后电流）；$X = 0$ 阻性（谐振）。

正弦稳态下，负载获得最大功率的条件是共轭匹配：

$$Z_L = Z_{th}^* \quad \text{即 } R_L = R_{th}, \; X_L = -X_{th}$$

📝 真题详解：正弦稳态功率计算

① 求电流：

② 阻抗角：$\varphi = \arctan(8/6) = 53.13°$

③ 有功功率：$P = UI\cos\varphi = 100 \times 10 \times \cos 53.13° = 1000 \times 0.6 = 600W$

④ 无功功率：$Q = UI\sin\varphi = 100 \times 10 \times \sin 53.13° = 1000 \times 0.8 = 800\text{var}$

⑤ 视在功率：$S = UI = 100 \times 10 = 1000\text{V·A}$

验证：$S = \sqrt{P^2+Q^2} = \sqrt{600^2+800^2} = \sqrt{360000+640000} = \sqrt{1000000} = 1000$ ✓

⑥ 功率因数：$\cos\varphi = 0.6$（滞后，感性）

• 电流从同名端流入 → 互感电压在同名端为正（同号）

• 电流从异名端流入 → 互感电压取负号

• 耦合系数 $k = M/\sqrt{L_1 L_2}$，$0 \le k \le 1$。$k=1$ 为全耦合（理想变压器）

理想变压器

全耦合（$k=1$）、无损耗、自感无穷大的变压器。变比 $n = N_1/N_2$。

$$\frac{u_1}{u_2} = n, \quad \frac{i_1}{i_2} = \frac{1}{n}, \quad Z_{in} = n^2 Z_L$$

即：电压比等于匝数比，电流比等于匝数反比，从原边看进去的阻抗等于副边负载乘 $n^2$。

📝 真题详解：对称三相Y型电路功率计算

对称三相Y型负载，每相 $Z = 8 + j6\,\Omega$，线电压 $U_l = 380V$。求线电流、有功功率、无功功率、功率因数。

✅ 完整解答

① 相电压：

$$U_p = \frac{U_l}{\sqrt{3}} = \frac{380}{\sqrt{3}} = 220V$$

② 阻抗模与阻抗角：

$$|Z| = \sqrt{8^2+6^2} = 10\Omega, \quad \varphi = \arctan\frac{6}{8} = 36.87°$$

③ 相电流 = 线电流（Y型 $I_l = I_p$）：

$$I_l = I_p = \frac{U_p}{|Z|} = \frac{220}{10} = 22A$$

④ 功率因数：

$$\cos\varphi = \frac{R}{|Z|} = \frac{8}{10} = 0.8 \text{（滞后）}$$

⑤ 有功功率：

$$P = \sqrt{3}U_lI_l\cos\varphi = \sqrt{3} \times 380 \times 22 \times 0.8 \approx 11584W \approx 11.58kW$$

验证：$P = 3 \times I_p^2 \times R = 3 \times 22^2 \times 8 = 3 \times 484 \times 8 = 11616W$ ✓（微小误差为计算取整）

⑥ 无功功率：

$$Q = \sqrt{3}U_lI_l\sin\varphi = \sqrt{3} \times 380 \times 22 \times 0.6 \approx 8688\text{var} \approx 8.69k\text{var}$$

📝 综合题：含受控源的正弦稳态分析

正弦电路中 $\dot{U}_S = 10\angle 0° V$，$R=4\Omega$，$X_L=3\Omega$，VCCS：$\dot{I}_c = 2\dot{U}_1$（$\dot{U}_1$ 为 $R$ 两端电压），VCCS 与 $R$ 并联后串 $X_L$ 接 $\dot{U}_S$。求总电流 $\dot{I}$ 和功率因数。

✅ 完整解答

① 设总电流 $\dot{I}$（流过 $X_L$ 的电流）：

$\dot{U}_1 = \dot{I} \times (R \parallel \text{VCCS支路})$

VCCS电流：$\dot{I}_c = 2\dot{U}_1 = 2 \times 4\dot{I}_R$（$\dot{I}_R$ 为流过 $R$ 的电流）

② KCL 在节点上：

$\dot{I} = \dot{I}_R + \dot{I}_c = \dot{I}_R + 2 \times 4\dot{I}_R = \dot{I}_R + 8\dot{I}_R = 9\dot{I}_R$

所以 $\dot{I}_R = \dot{I}/9$，$\dot{U}_1 = R \times \dot{I}_R = 4\dot{I}/9$

③ 等效阻抗：

$\dot{U}_{\text{并联}} = \dot{U}_1 = \frac{4}{9}\dot{I}$，所以 $Z_{\text{并联}} = 4/9 \,\Omega$

$Z_{total} = Z_{\text{并联}} + jX_L = \frac{4}{9} + j3 \,\Omega$

④ 求总电流：

$$\dot{I} = \frac{\dot{U}_S}{Z_{total}} = \frac{10\angle 0°}{0.444 + j3} = \frac{10}{3.033\angle 81.6°} = 3.30\angle(-81.6°) A$$

⑤ 功率因数：$\cos\varphi = \cos 81.6° \approx 0.146$（非常低，因为受控源使等效电阻极小）

💡 综合题策略：遇到含受控源+正弦稳态的综合题，先用相量法将所有量转为复数，再按直流电路的方法（节点/网孔）列方程，最后补充受控源控制量方程。计算时注意复数运算的准确性。

常见失分点（6大陷阱）

① 叠加定理算功率 — 功率是平方关系，不能叠加！$P=(I_1+I_2)^2R \neq I_1^2R+I_2^2R$

② 三相功率漏 √3 — $P=\sqrt{3}U_lI_l\cos\varphi$，公式中的 $\sqrt{3}$ 最容易忘！

③ 受控源置零 — 受控源永远不能置零！叠加定理和求 $R_{th}$ 时都要保留。

④ 功率因数角搞错 — $\varphi$ 是相电压与相电流的相位差（每相阻抗角），不是线电压与线电流的相位差。

⑤ 换路定则用错对象 — $u_C$ 和 $i_L$ 不能突变，但 $i_C$ 和 $u_L$ 可以突变！0+ 等效电路中电容→电压源、电感→电流源。

⑥ 运放虚短条件忽略 — 虚短前提是线性区（有负反馈）。如果没有负反馈，不能使用虚短！
